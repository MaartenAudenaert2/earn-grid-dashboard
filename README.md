# Earn Grid

Publieke, read-only momentopname van de persoonlijke opportunity monitor.

De pagina toont alleen geselecteerde, niet-geheime gegevens:

- gevalideerde kansen en laatste scanresultaten;
- volgende geplande scans;
- bevestigd inkomen tegenover het maanddoel;
- geaggregeerd Taskmarket-USDC-saldo;
- publieke GitHub PR-status en geanonimiseerde inzendstatus.

## Privacy en accounting

- Geen walletadres, private key, API-token, lokaal pad of secret wordt gepubliceerd.
- Een walletsaldo is geen inkomen.
- Alleen aantoonbaar ontvangen betalingen tellen mee voor het maanddoel.
- Pending beloningen of inzendingen worden expliciet niet als verdiend geteld.

De data wordt vanuit de lokale read-only monitor als beperkte snapshot gepubliceerd.
